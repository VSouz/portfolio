Meus projetos

VOCÁBULO

O projeto Vocábulo é inspirado no site term.ooo e foi uma excelente oportunidade para praticar minhas habilidades em HTML, CSS e JavaScript. Como sou fã do jogo, pensei: por que não me desafiar a criar minha própria versão? Assim, pude experimentar e entender mais sobre lógica de jogos e interatividade na web.

CAFETERIA.JAVA

Esse projeto foi desenvolvido para a disciplina Linguagem de Programação 2. A proposta era criar um sistema com interface gráfica em um curto prazo. Escolhi desenvolver um sistema de autoatendimento para uma cafeteria.

EPIC FIGHT

Epic Fight é um jogo platformer emocionante desenvolvido com as bibliotecas permitidas pgzero, random e o objeto React do pygame pois se trata de um desafio para uma vaga de emprego. O objetivo do jogo é simples: derrotar o boss enquanto navega por vários obstáculos e desafios.

YT SAVE

Ytsave foi um projeto pessoal que desenvolvi para simplificar o download rápido de vídeos e músicas do YouTube. Nele, utilizei o Flask pela primeira vez para criar uma interface simples e eficiente, explorando as funcionalidades da biblioteca e aprimorando minhas habilidades em desenvolvimento web.
